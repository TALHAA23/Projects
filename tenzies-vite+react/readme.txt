-About

Tenzies is a game which have dice and the goal is to roll all the dice until all the dice have same face.
With each roll dice of same face can be freeze so it doesn't change with rolling.

-Features

.Responsive UI
.update UI thanks to React
.count rolls
.uptime of each play
.store best score in localStorage

-Installation

.download files
.open src file directory in an editor like VS code
.head to project directory
.open terminal and install packages using npm install
.in the termail run npm run dev or yarn run dev
